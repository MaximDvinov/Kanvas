# Security Policy

## Reporting

If you discover a security issue, do not open a public issue.

Please report privately with:
- impacted module(s);
- reproduction steps;
- potential severity;
- suggested remediation if available.

## Response Targets

- Initial response: within 5 business days.
- Triage decision: within 10 business days.
- Patch timeline: depends on severity and platform impact.

## Supported Versions

Until semantic versioning is introduced, only the latest `main` commit is
considered supported for security fixes.
